Value

Value is the lightness or darkness of a color.

Value obviously relates to every color in the color spectrum. Value can be represented by numbers. 1 is the darkest value (low value). 9 is the lightest value (high value). Value has a lot to do with creating contrast and depth. Thanks to value we can detect a light source and recognize a 3-dimensional shape.
